```bash
# prerequirements
$ npm i -g tooling

# dev mode
$ npm run dev

$ npm run deploy
```

## Known Issues

On Safari, You have to press <kbd>Command+S</kbd> or <kbd>Ctrl+S</kbd> to export as JSON file in the opened page.

On Safari and IE, `input` types `date` and `time` are not supported in `/add` page.
